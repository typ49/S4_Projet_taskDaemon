# S4_Projet_taskDaemon

## Compilation :

__To install  the "libmessage.so" library :__
```sh
make install
```

__To compil every programs of the project :__ 
```sh
make
```

__To compil a specific program :__
```sh
make <program-name>
```

__To clean the compiled programs :__
```sh
make clean
```
